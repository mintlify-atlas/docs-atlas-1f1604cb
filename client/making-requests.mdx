The ZIO HTTP client provides multiple ways to construct and execute HTTP requests, from simple one-liners to complex request building with custom headers, query parameters, and bodies.

## Basic Request Methods

### Using Client Service Methods

The most convenient way to make requests is using the client's HTTP verb methods:

```scala
import zio._
import zio.http._

val program = for {
  client <- ZIO.service[Client]
  
  // GET request
  getResponse <- client.url(baseUrl).get("/users")
  
  // POST request with body
  postResponse <- client.url(baseUrl).post("/users")(userBody)
  
  // PUT request
  putResponse <- client.url(baseUrl).put("/users/1")(updatedUser)
  
  // DELETE request
  deleteResponse <- client.url(baseUrl).delete("/users/1")
  
  // PATCH request
  patchResponse <- client.url(baseUrl).patch("/users/1")
  
  // HEAD request
  headResponse <- client.url(baseUrl).head("/users")
} yield ()
```

### Using Request Objects

For more control, construct `Request` objects directly:

```scala
// GET request
val getRequest = Request.get("/users")

// POST request with body
val postRequest = Request.post("/users", Body.fromString(jsonBody))

// PUT request
val putRequest = Request.put("/users/1", Body.fromString(jsonBody))

// DELETE request
val deleteRequest = Request.delete("/users/1")

// Execute the request
val response: ZIO[Client, Throwable, Response] = 
  ZClient.batched(getRequest)
```

## Building URLs

### Decoding from String

```scala
// Decode from string
val url: Either[MalformedURLException, URL] = 
  URL.decode("https://api.example.com/users?page=1")

// Use with ZIO
val program = for {
  url <- ZIO.fromEither(URL.decode("https://api.example.com"))
  response <- ZClient.batched(Request.get(url))
} yield response
```

## Working with Headers

### Adding Headers to Client

```scala
val client = ZIO.serviceWith[Client] { client =>
  client
    .url(baseUrl)
    .addHeader(Header.Authorization.Bearer("your-token"))
    .addHeader(Header.Accept(MediaType.application.json))
    .addHeader(Header.Custom("X-API-Key", "key123"))
}
```

### Adding Headers to Requests

```scala
// Using Request builder
val request = Request.get(url)
  .addHeader(Header.Authorization.Bearer("token"))
  .addHeader(Header.ContentType(MediaType.application.json))

// Execute with custom headers
val response = for {
  client <- ZIO.service[Client]
  res <- client.url(baseUrl).addHeaders(customHeaders).get("/users")
} yield res
```

### Header Operations

The client and request both implement `HeaderOps` for convenient header manipulation:

```scala
// Update headers functionally
val updatedClient = client.updateHeaders { headers =>
  headers
    .addHeader(Header.UserAgent("MyApp/1.0"))
    .removeHeader(Header.Accept)
}
```

## Working with Request Bodies

### String Body

```scala
val jsonBody = Body.fromString("""{ "name": "John", "email": "john@example.com" }""")
val request = Request.post("/users", jsonBody)
```

### Bytes Body

```scala
val bytesBody = Body.fromChunk(Chunk.fromArray(data))
val request = Request.post("/upload", bytesBody)
```

### Form Data

```scala
val formBody = Body.fromURLEncodedForm(
  Form(
    FormField.simpleField("username", "john"),
    FormField.simpleField("password", "secret")
  )
)
val request = Request.post("/login", formBody)
```

### Updating Request Body

```scala
// Replace body
val updatedRequest = request.withBody(newBody)

// Transform body
val transformedRequest = request.updateBody { body =>
  // Modify body
  body
}

// Transform body effectfully
val effectfulUpdate = request.updateBodyZIO { body =>
  // Perform effects
  ZIO.succeed(transformedBody)
}
```

## Executing Requests

### Batched Requests (Fully Materialized)

```scala
// Using ZClient.batched - materializes response in memory
val response: ZIO[Client, Throwable, Response] = 
  ZClient.batched(Request.get(url))

// Using client instance
val program = for {
  client <- ZIO.service[Client]
  response <- client.batched(Request.get(url))
  body <- response.body.asString
} yield body
```

Reference: [ZClient.scala:295](zio-http/shared/src/main/scala/zio/http/ZClient.scala:295)

<Note>
**Note**: Batched requests materialize the full response into memory before resuming. This is convenient but not suitable for large response bodies.
</Note>

### Streaming Requests

```scala
// Requires Scope for resource management
val streamingProgram = ZIO.scoped {
  for {
    response <- ZClient.streaming(Request.get(url))
    // Process streaming body
    _ <- response.body.asStream.foreach { chunk =>
      // Process chunk
      ZIO.succeed(println(s"Received ${chunk.size} bytes"))
    }
  } yield ()
}
```

Reference: [ZClient.scala:328](zio-http/shared/src/main/scala/zio/http/ZClient.scala:328)

<Note>
**Important**: When using streaming mode, you must close the Scope **after** the body has been fully consumed. The response body is only valid within the Scope.
</Note>

### Stream Transformation

```scala
// Transform response to stream
val stream: ZStream[Client, Throwable, String] = 
  ZClient.streamingWith(Request.get(url)) { response =>
    response.body.asStream.via(ZPipeline.utf8Decode)
  }

// Using client instance
val program = for {
  client <- ZIO.service[Client]
  stream = client.stream(Request.get(url)) { response =>
    response.body.asStream.map(_.asString)
  }
  lines <- stream.runCollect
} yield lines
```

## Complete Example

```scala
import zio._
import zio.http._

object CompleteExample extends ZIOAppDefault {
  val program = for {
    // Build URL
    url <- ZIO.fromEither(
      URL.decode("https://jsonplaceholder.typicode.com")
    )
    
    // Get client
    client <- ZIO.service[Client]
    
    // Configure client
    configuredClient = client
      .url(url)
      .addPath("/posts")
      .addQueryParam("userId", "1")
      .addHeader(Header.Accept(MediaType.application.json))
    
    // Make batched request
    response <- configuredClient.batched(Request.get("/"))
    
    // Process response
    body <- response.body.asString
    _ <- Console.printLine(s"Status: ${response.status}")
    _ <- Console.printLine(s"Body: ${body}")
  } yield ()
  
  override val run = program.provide(Client.default)
}
```
