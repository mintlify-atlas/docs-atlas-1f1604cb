This guide will walk you through creating a simple HTTP server and client with ZIO HTTP. You'll learn the basics of routing, handlers, and how to test your endpoints.

## Your First Server

Let's create a greeting server with two routes: a root endpoint and a personalized greeting endpoint.

<Steps>

<Step title="Create the Server">

Create a new Scala file and add the following code:

```scala
import zio._
import zio.http._

object GreetingServer extends ZIOAppDefault {
  val routes =
    Routes(
      Method.GET / Root -> handler(Response.text("Greetings at your service")),
      Method.GET / "greet" -> handler { (req: Request) =>
        val name = req.queryOrElse[String]("name", "World")
        Response.text(s"Hello $name!")
      }
    )

  def run = Server.serve(routes).provide(Server.default)
}
```

This server defines two routes:
- `GET /` - Returns a welcome message
- `GET /greet?name=<name>` - Returns a personalized greeting

</Step>

<Step title="Run the Server">

Start your server using your build tool:

<CodeGroup>

```bash sbt
sbt "runMain GreetingServer"
```

```bash Maven
mvn compile exec:java -Dexec.mainClass="GreetingServer"
```

```bash Gradle
gradle run
```

</CodeGroup>

The server will start on `http://localhost:8080` by default.

</Step>

<Step title="Test the Endpoints">

Open a new terminal and test your endpoints:

```bash
# Test the root endpoint
curl http://localhost:8080/
# Output: Greetings at your service

# Test the greeting endpoint
curl http://localhost:8080/greet?name=John
# Output: Hello John!

# Test with default name
curl http://localhost:8080/greet
# Output: Hello World!
```

</Step>

</Steps>

## Creating a Client

Now let's create a client to interact with the server programmatically:

```scala
import zio._
import zio.http._

object GreetingClient extends ZIOAppDefault {

  val app =
    for {
      client   <- ZIO.serviceWith[Client](_.host("localhost").port(8080))
      request  =  Request.get("greet").addQueryParam("name", "John")
      response <- client.batched(request)
      _        <- response.body.asString.debug("Response")
    } yield ()

  def run = app.provide(Client.default)
}
```

<Note>
Make sure your `GreetingServer` is running before executing the client!
</Note>

Run the client:

```bash
sbt "runMain GreetingClient"
```

You should see output similar to:

```
Response: Hello John!
```

## Understanding the Code

Let's break down the key concepts:

### Routes

Routes define the mapping between HTTP requests and handlers:

```scala
val routes = Routes(
  Method.GET / Root -> handler(Response.text("Hello")),
  Method.GET / "users" / int("id") -> handler { (id: Int, req: Request) =>
    Response.text(s"User $id")
  }
)
```

### Handlers

Handlers process requests and return responses:

```scala
// Simple handler
handler(Response.text("Static response"))

// Handler with request access
handler { (req: Request) =>
  val param = req.queryOrElse[String]("key", "default")
  Response.text(s"Param: $param")
}

// Handler with path parameters
handler { (id: Int, req: Request) =>
  Response.text(s"ID: $id")
}
```

### Responses

Create different types of responses:

```scala
// Text response
Response.text("Hello World!")

// JSON response
Response.json("""{"message": "Hello"}""")

// Status code response
Response.status(Status.NotFound)

// Response with headers
Response.text("Hello").addHeader(Header.ContentType(MediaType.text.plain))
```

## A More Complete Example

Here's a more realistic example with multiple routes and JSON handling:

```scala
import zio._
import zio.http._

object HelloWorld extends ZIOAppDefault {
  // Responds with plain text
  val homeRoute =
    Method.GET / Root -> handler(Response.text("Hello World!"))

  // Responds with JSON
  val jsonRoute =
    Method.GET / "json" -> handler(Response.json("""{"greetings": "Hello World!"}"""))

  // Create HTTP route
  val app = Routes(homeRoute, jsonRoute)

  // Run it like any simple app
  override val run = Server.serve(app).provide(Server.default)
}
```

## Configuration Options

### Custom Port

Change the default port (8080) by providing a custom server configuration:

```scala
import zio.http._

val config = Server.Config.default.port(9000)

Server.serve(routes).provide(Server.live(config))
```

### Server with Multiple Settings

Configure various server options:

```scala
val config = Server.Config.default
  .port(9000)
  .ssl(sslConfig)
  .maxHeaderSize(8192)

Server.serve(routes).provide(Server.live(config))
```

## Common Patterns

### Query Parameters

Extract query parameters from requests:

```scala
Method.GET / "search" -> handler { (req: Request) =>
  val query = req.queryOrElse[String]("q", "")
  val page = req.queryOrElse[Int]("page", 1)
  Response.text(s"Searching for: $query, page: $page")
}
```

### Path Parameters

Use path parameters with type safety:

```scala
Method.GET / "users" / int("userId") -> handler { (userId: Int, req: Request) =>
  Response.text(s"User ID: $userId")
}

Method.GET / "posts" / string("slug") -> handler { (slug: String, req: Request) =>
  Response.text(s"Post: $slug")
}
```

### Request Body

Handle request bodies:

```scala
Method.POST / "users" -> handler { (req: Request) =>
  for {
    body <- req.body.asString
    _    <- Console.printLine(s"Received: $body")
  } yield Response.text("User created")
}
```

## Error Handling

Handle errors gracefully in your routes:

```scala
Method.GET / "risky" -> handler {
  ZIO.attempt {
    // Some operation that might fail
    throw new RuntimeException("Something went wrong")
  }.catchAll { error =>
    ZIO.succeed(Response.text(error.getMessage).status(Status.InternalServerError))
  }
}
```

## Testing Your Server

ZIO HTTP provides a testkit for testing without a live server. Add the dependency:

```scala
libraryDependencies += "dev.zio" %% "zio-http-testkit" % "3.3.2" % Test
```

Then write tests:

```scala
import zio.test._
import zio.http._

object GreetingServerSpec extends ZIOSpecDefault {
  def spec = suite("GreetingServer")(
    test("root endpoint returns greeting") {
      val request = Request.get(URL.root)
      val response = routes.run(request)
      
      for {
        res <- response
        body <- res.body.asString
      } yield assertTrue(body == "Greetings at your service")
    }
  )
}
```
