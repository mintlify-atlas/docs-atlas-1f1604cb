`Request` represents an HTTP request with method, URL, headers, and body.

## Type Signature

```scala
final case class Request(
  version: Version = Version.Default,
  method: Method = Method.ANY,
  url: URL = URL.empty,
  headers: Headers = Headers.empty,
  body: Body = Body.empty,
  remoteAddress: Option[InetAddress] = None,
  remoteCertificate: Option[Certificate] = None,
)
```

<ParamField path="version" type="Version" default="Version.Default">
  The HTTP protocol version
</ParamField>

<ParamField path="method" type="Method" default="Method.ANY">
  The HTTP method (GET, POST, etc.)
</ParamField>

<ParamField path="url" type="URL" default="URL.empty">
  The request URL containing path and query parameters
</ParamField>

<ParamField path="headers" type="Headers" default="Headers.empty">
  HTTP headers
</ParamField>

<ParamField path="body" type="Body" default="Body.empty">
  The request body
</ParamField>

<ParamField path="remoteAddress" type="Option[InetAddress]" default="None">
  The client's IP address
</ParamField>

<ParamField path="remoteCertificate" type="Option[Certificate]" default="None">
  Client certificate for mutual TLS
</ParamField>

## Modifying Requests

### Combining Requests

```scala
def ++(that: Request): Request
```

Combines two requests. Information from `that` takes precedence in conflicts.

<ParamField path="that" type="Request">
  The request to merge with
</ParamField>

**Example:**

```scala
val baseReq = Request.get("/api")
val withAuth = Request(headers = Headers("Authorization", "Bearer token"))
val combined = baseReq ++ withAuth
```
