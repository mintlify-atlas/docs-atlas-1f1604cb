`Response` represents an HTTP response with status code, headers, and body.

## Type Signature

```scala
final case class Response(
  status: Status = Status.Ok,
  headers: Headers = Headers.empty,
  body: Body = Body.empty,
)
```

<ParamField path="status" type="Status" default="Status.Ok">
  The HTTP status code
</ParamField>

<ParamField path="headers" type="Headers" default="Headers.empty">
  HTTP response headers
</ParamField>

<ParamField path="body" type="Body" default="Body.empty">
  The response body
</ParamField>

## Creating Responses

### Response.ok

```scala
val ok: Response
```

Creates an empty response with 200 OK status.

### Response.status

```scala
def status(status: Status): Response
```

Creates an empty response with the specified status.

<ParamField path="status" type="Status">
  The HTTP status code
</ParamField>

**Example:**

```scala

```

### Response.text

```scala

```

Creates a response with plain text content.

<ParamField path="text" type="CharSequence">

</ParamField>

**Example:**

```scala

```

### Response.html

```scala

```

Creates a response with HTML content.

<ParamField path="data" type="Html">
  The HTML template
</ParamField>

<ParamField path="status" type="Status" default="Status.Ok">
  The HTTP status
</ParamField>

**Example:**

```scala

```

### Response.json

```scala

```

Creates a response with JSON content (sets Content-Type to application/json).

<ParamField path="data" type="CharSequence">

**Example:**

```scala

```

## Creating from Effects

### Response.fromCause

```scala
def fromCause(cause: Cause[Any]): Response
def fromCause(cause: Cause[Any], config: ErrorResponseConfig): Response
```

Creates a response from a ZIO Cause, automatically determining the status code.

<ParamField path="cause" type="Cause[Any]">
  The error cause
</ParamField>

<ParamField path="config" type="ErrorResponseConfig">
  Configuration for error responses
</ParamField>

**Example:**

```scala
Response.fromCause(Cause.fail(new Exception("Database error")))
// Returns 500 Internal Server Error
```

### Response.fromCauseWith

```scala
def fromCauseWith[E](cause: Cause[E], config: ErrorResponseConfig)(f: E => Response): Response
```

Creates a response from a Cause, using a function to handle typed errors.

<ParamField path="f" type="E => Response">
  Function to convert typed errors to responses
</ParamField>

### Response.fromThrowable

```scala
def fromThrowable(throwable: Throwable): Response
def fromThrowable(throwable: Throwable, config: ErrorResponseConfig): Response
```

Creates a response from a Throwable with appropriate status code.

<ParamField path="throwable" type="Throwable">
  The exception
</ParamField>

Status code mapping:
- `FileNotFoundException` → 404 Not Found
- `AccessDeniedException` → 403 Forbidden
- `IllegalArgumentException` → 400 Bad Request
- `ConnectException` → 503 Service Unavailable
- `SocketTimeoutException` → 504 Gateway Timeout
- Other → 500 Internal Server Error

## Response.Patch

```scala
sealed trait Patch
```

Represents modifications to apply to a response.

### Creating Patches

#### Patch.empty

```scala
val empty: Patch
```

#### Patch.addHeader

```scala
def addHeader(name: CharSequence, value: CharSequence): Patch
def addHeader(header: Header): Patch
def addHeaders(headers: Headers): Patch
```

Creates a patch that adds headers.

#### Patch.removeHeaders

```scala
def removeHeaders(headerTypes: Set[HeaderType]): Patch
```

Creates a patch that removes headers.

#### Patch.status

```scala
def status(status: Status): Patch
```

Creates a patch that sets the status.

#### Patch.updateHeaders

```scala
def updateHeaders(f: Headers => Headers): Patch
```

Creates a patch that transforms headers.

### Combining Patches

```scala
def ++(that: Patch): Patch
```

Combines two patches.

**Example:**

```scala
val patch = 
  Response.Patch.status(Status.Created) ++
  Response.Patch.addHeader("X-Request-ID", "123")
  
response.patch(patch)
```

### Applying Patches

```scala
def apply(res: Response)(implicit trace: Trace): Response
```

Applies the patch to a response.

<ParamField path="res" type="Response">
  The response to modify
</ParamField>

<ResponseField name="return" type="Response">
  The modified response
</ResponseField>
