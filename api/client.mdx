The `ZClient` type represents an HTTP client for making requests to HTTP servers.

## Type Signature

```scala
final case class ZClient[
  -Env,
  ReqEnv,
  -In,
  +Err,
  +Out
](
  version: Version,
  url: URL,
  headers: Headers,
  sslConfig: Option[ClientSSLConfig],
  proxy: Option[Proxy],
  bodyEncoder: ZClient.BodyEncoder[Env, Err, In],
  bodyDecoder: ZClient.BodyDecoder[Env, Err, Out],
  driver: ZClient.Driver[Env, ReqEnv, Err]
)
```

## Creating Clients

### Client.default

```scala
val default: ZLayer[Any, Throwable, Client]
```

Creates a client with default configuration.

**Example:**
```scala
val app = ZIO.serviceWithZIO[Client] { client =>
  client(Request.get("https://api.example.com/users"))
}

app.provide(Client.default)
```

## Complete Example

```scala
import zio._
import zio.http._

object ClientExample extends ZIOAppDefault {
  val app = for {
    client <- ZIO.service[Client]
    
    // Configure client
    apiClient = client
      .host("api.example.com")
      .port(443)
      .ssl(ClientSSLConfig.Default)
      .addHeader("Authorization", "Bearer token")
    
    // Make request
    response <- apiClient.batched(
      Request.get("/users/123")
    )
    
    body <- response.body.asString
    _    <- Console.printLine(body)
  } yield ()
  
  def run = app.provide(Client.default)
}
```
