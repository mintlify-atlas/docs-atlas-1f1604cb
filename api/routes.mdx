title: Routes
description: API reference for zio.http.Routes
---

`Routes` is a collection of routes that forms an HTTP application. All errors must be handled through conversion into HTTP responses before routes can be executed.

## Type Signature

```scala
final case class Routes[-Env, +Err](routes: Chunk[zio.http.Route[Env, Err]])
```

<ParamField path="Env" type="type parameter">
  The environment type required by the routes
</ParamField>

<ParamField path="Err" type="type parameter">
  The error type that can be produced by the routes
</ParamField>

<ParamField path="routes" type="Chunk[Route[Env, Err]]">
  The collection of individual routes that make up this application
</ParamField>

### Static File Serving

#### Routes.serveDirectory

```scala
def serveDirectory(path: Path, docRoot: File)(implicit trace: Trace): Routes[Any, Nothing]
```

Creates routes for serving static files from a local directory.

<ParamField path="path" type="Path">
  The URL path prefix where files will be served
</ParamField>

<ParamField path="docRoot" type="File">
  The local directory containing the files to serve
</ParamField>

**Example:**

```scala
Routes.serveDirectory(Path.empty / "assets", new File("/var/www/static"))
// Request to /assets/images/logo.png serves /var/www/static/images/logo.png
```

#### Routes.serveResources

```scala
def serveResources(path: Path, resourcePrefix: String = "public")(implicit trace: Trace): Routes[Any, Nothing]
```

Creates routes for serving static files from the classpath resources.

<ParamField path="path" type="Path">
  The URL path prefix where resources will be served
</ParamField>

<ParamField path="resourcePrefix" type="String" default="public">
  The resource directory prefix in src/main/resources
</ParamField>

## Combining Routes

### ++

```scala
def ++[Env1 <: Env, Err1 >: Err](that: Routes[Env1, Err1]): Routes[Env1, Err1]
```

Combines two Routes. In case of conflicts, routes from `that` take precedence.

<ParamField path="that" type="Routes[Env1, Err1]">
  The routes to append
</ParamField>

### +: and :+

```scala
def +:[Env1 <: Env, Err1 >: Err](route: Route[Env1, Err1]): Routes[Env1, Err1]
def :+[Env1 <: Env, Err1 >: Err](route: Route[Env1, Err1]): Routes[Env1, Err1]
```

Prepend or append a single route to the routes collection.

## Error Handling

### handleError

```scala
def handleError(f: Err => Response)(implicit trace: Trace): Routes[Env, Nothing]
```

Handles all typed errors by converting them into responses.

<ParamField path="f" type="Err => Response">
  Function to convert errors into HTTP responses
</ParamField>

### handleErrorZIO

```scala
def handleErrorZIO[Env1 <: Env](f: Err => ZIO[Env1, Nothing, Response])(implicit trace: Trace): Routes[Env1, Nothing]
```

Handles errors effectfully, producing responses via ZIO.

### handleErrorCause

```scala
def handleErrorCause(f: Cause[Err] => Response)(implicit trace: Trace): Routes[Env, Nothing]
```

Handles both typed errors and non-recoverable errors (defects) by converting the full cause into a response.

### handleErrorCauseZIO

```scala
def handleErrorCauseZIO(f: Cause[Err] => ZIO[Any, Nothing, Response])(implicit trace: Trace): Routes[Env, Nothing]
```

Effectful version that handles all errors and defects.

### handleErrorRequest

```scala
def handleErrorRequest(f: (Err, Request) => Response)(implicit trace: Trace): Routes[Env, Nothing]
```

Handles errors with access to the request that caused the error.

<ParamField path="f" type="(Err, Request) => Response">
  Function receiving both the error and the originating request
</ParamField>

### handleErrorRequestCause

```scala
def handleErrorRequestCause(f: (Request, Cause[Err]) => Response)(implicit trace: Trace): Routes[Env, Nothing]
```

Handles error causes with access to the originating request.

### sandbox

```scala
def sandbox(implicit trace: Trace): Routes[Env, Nothing]
```

Automatically translates all failures into responses using best-effort heuristics for HTTP status codes.

## Server Integration

### serve

```scala
def serve[Env1 <: Env](implicit
  ev: Err <:< Response,
  trace: Trace,
  tag: EnvironmentTag[Env1],
): URIO[Env1 with Server, Nothing]
```

Installs routes into a server and runs forever.

<ResponseField name="return" type="URIO[Env1 with Server, Nothing]">
  Effect that installs routes and serves requests indefinitely
</ResponseField>

**Example:**

```scala
val app = Routes(
  Method.GET / "hello" -> handler(Response.text("Hello World"))
)

app.serve // Requires Server in environment
```
