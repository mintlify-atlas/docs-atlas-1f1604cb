title: Handler
description: API reference for zio.http.Handler
---

`Handler` is a functional abstraction for processing input into output, with support for errors and environmental requirements. It forms the core building block for request processing in ZIO HTTP.

## Type Signature

```scala
sealed trait Handler[-R, +Err, -In, +Out]
```

<ParamField path="R" type="type parameter">
  The environment type required by the handler
</ParamField>

<ParamField path="Err" type="type parameter">
  The error type that can be produced
</ParamField>

<ParamField path="In" type="type parameter">
  The input type accepted by the handler
</ParamField>

<ParamField path="Out" type="type parameter">
  The output type produced by the handler
</ParamField>

## Core Operation

### apply

```scala
def apply(in: In): ZIO[Scope & R, Err, Out]
```

Executes the handler with the provided input.

<ParamField path="in" type="In">
  The input value to process
</ParamField>

<ResponseField name="return" type="ZIO[Scope & R, Err, Out]">
  Effect that produces the output or fails with an error
</ResponseField>

## Creating Handlers

### Handler.succeed

```scala
def succeed[Out](out: => Out): Handler[Any, Nothing, Any, Out]
```

Creates a handler that always succeeds with the same value.

<ParamField path="out" type="Out">
  The value to always return
</ParamField>

### Handler.fail

```scala
def fail[Err](err: => Err): Handler[Any, Err, Any, Nothing]
```

Creates a handler that always fails with the specified error.

### Handler.fromZIO

```scala
def fromZIO[R, Err, Out](zio: => ZIO[R, Err, Out]): Handler[R, Err, Any, Out]
```

Converts a ZIO effect into a handler.

<ParamField path="zio" type="ZIO[R, Err, Out]">
  The ZIO effect to convert
</ParamField>

**Example:**

```scala
Handler.fromZIO(Random.nextInt)
```

### Handler.fromFunction

```scala
def fromFunction[In]: FromFunction[In]
```

Creates a handler from a pure function `In => Out`.

**Example:**

```scala
Handler.fromFunction[String] { str => str.toUpperCase }
```

### Handler.fromFunctionZIO

```scala
def fromFunctionZIO[In]: FromFunctionZIO[In]
```

Creates a handler from an effectful function `In => ZIO[R, Err, Out]`.

**Example:**

```scala
Handler.fromFunctionZIO[Request] { req =>
  database.fetchUser(req.header("user-id"))
}
```

### Handler.fromFunctionHandler

```scala
def fromFunctionHandler[In]: FromFunctionHandler[In]
```

Creates a handler from a function that returns a handler.

### Handler.attempt

```scala
def attempt[Out](out: => Out): Handler[Any, Throwable, Any, Out]
```

Attempts to create a handler, catching any non-fatal exceptions.

### Handler.identity

```scala
def identity[A]: Handler[Any, Nothing, A, A]
```

Pass-through handler that returns its input unchanged.

## Transformation

```scala
def map[Out1](f: Out => Out1)(implicit trace: Trace): Handler[R, Err, In, Out1]
```

<ParamField path="f" type="Out => Out1">

### mapZIO

```scala
def mapZIO[R1 <: R, Err1 >: Err, Out1](f: Out => ZIO[R1, Err1, Out1])(implicit trace: Trace): Handler[R1, Err1, In, Out1]
```

### contramap

```scala
def contramap[In1](f: In1 => In): Handler[R, Err, In1, Out]
```

<ParamField path="f" type="In1 => In">
  Function to transform the input
</ParamField>

### contramapZIO

```scala
def contramapZIO[R1 <: R, Err1 >: Err, In1](f: In1 => ZIO[R1, Err1, In])(implicit trace: Trace): Handler[R1, Err1, In1, Out]
```

### flatMap

```scala
def flatMap[R1 <: R, Err1 >: Err, In1 <: In, Out1](f: Out => Handler[R1, Err1, In1, Out1])(implicit trace: Trace): Handler[R1, Err1, In1, Out1]
```

```scala
def flatten[R1 <: R, Err1 >: Err, In1 <: In, Out1](implicit ev: Out <:< Handler[R1, Err1, In1, Out1], trace: Trace): Handler[R1, Err1, In1, Out1]
```

## Composition

### Right Composition Operator

```scala
def >>>[R1 <: R, Err1 >: Err, In1 >: Out, Out1](that: Handler[R1, Err1, In1, Out1])(implicit trace: Trace): Handler[R1, Err1, In, Out1]
```

Composes handlers left-to-right using the `>>>` operator (pipes output into next handler).

**Example:**

```scala
val pipeline = parseJson >>> validateData >>> saveToDatabase
```

### Left Composition Operator

```scala
def <<<[R1 <: R, Err1 >: Err, In1, Out1 <: In](that: Handler[R1, Err1, In1, Out1])(implicit trace: Trace): Handler[R1, Err1, In1, Out]
```

Composes handlers right-to-left using the `<<<` operator.

### andThen / compose

```scala
def andThen[R1 <: R, Err1 >: Err, In1 >: Out, Out1](that: Handler[R1, Err1, In1, Out1])(implicit trace: Trace): Handler[R1, Err1, In, Out1]
def compose[R1 <: R, Err1 >: Err, In1, Out1 <: In](that: Handler[R1, Err1, In1, Out1])(implicit trace: Trace): Handler[R1, Err1, In1, Out]
```

Named aliases for `>>>` and `<<<`.

## Error Handling

### catchAll

```scala
def catchAll[R1 <: R, Err1, In1 <: In, Out1 >: Out](f: Err => Handler[R1, Err1, In1, Out1])(implicit trace: Trace): Handler[R1, Err1, In1, Out1]
```

Recovers from all errors using a function that produces a new handler.

<ParamField path="f" type="Err => Handler[R1, Err1, In1, Out1]">
  Function to handle errors
</ParamField>

### catchSome

```scala
def catchSome[R1 <: R, Err1 >: Err, In1 <: In, Out1 >: Out](pf: PartialFunction[Err, Handler[R1, Err1, In1, Out1]])(implicit trace: Trace): Handler[R1, Err1, In1, Out1]
```

Recovers from some errors using a partial function.

### mapError

```scala
def mapError[Err1](f: Err => Err1)(implicit trace: Trace): Handler[R, Err1, In, Out]
```

Transforms the error type.

### mapErrorZIO

```scala
def mapErrorZIO[R1 <: R, Err1, Out1 >: Out](f: Err => ZIO[R1, Err1, Out1])(implicit trace: Trace): Handler[R1, Err1, In, Out1]
```

Effectfully transforms errors.

### mapErrorCause

```scala
def mapErrorCause[Err2](f: Cause[Err] => Err2)(implicit trace: Trace): Handler[R, Err2, In, Out]
```

Transforms error causes (including defects).

### orElse

```scala
def orElse[R1 <: R, Err1, In1 <: In, Out1 >: Out](that: Handler[R1, Err1, In1, Out1])(implicit trace: Trace): Handler[R1, Err1, In1, Out1]
```

Runs the alternative handler if this one fails.

**Example:**

```scala
primaryHandler <> fallbackHandler
```

### orDie

```scala
def orDie(implicit ev1: Err <:< Throwable, ev2: CanFail[Err], trace: Trace): Handler[R, Nothing, In, Out]
```

Converts failures into defects (unchecked errors).

### sandbox

```scala
def sandbox(implicit trace: Trace): Handler[R, Response, In, Out]
```

Automatically converts all failures (including defects) into HTTP responses.

## Observation

### tapZIO

```scala
def tapZIO[R1 <: R, Err1 >: Err](f: Out => ZIO[R1, Err1, Any])(implicit trace: Trace): Handler[R1, Err1, In, Out]
```

Effectfully observes the output without changing it.

**Example:**

```scala
handler.tapZIO { response =>
  ZIO.logInfo(s"Status: ${response.status}")
}
```

### tapErrorZIO

```scala
def tapErrorZIO[R1 <: R, Err1 >: Err](f: Err => ZIO[R1, Err1, Any])(implicit trace: Trace): Handler[R1, Err1, In, Out]
```

Effectfully observes errors without changing them.

## Combining Handlers

### zip

```scala
def zip[R1 <: R, Err1 >: Err, In1 <: In, Out1](that: Handler[R1, Err1, In1, Out1])(implicit trace: Trace): Handler[R1, Err1, In1, (Out, Out1)]
```

Combines two handlers, producing a tuple of outputs.

### zipLeft

```scala
def zipLeft[R1 <: R, Err1 >: Err, In1 <: In, Out1](that: Handler[R1, Err1, In1, Out1])(implicit trace: Trace): Handler[R1, Err1, In1, Out]
```

Runs both handlers, keeping only the first output. Also available as the `<*` operator.

### zipRight

```scala
def zipRight[R1 <: R, Err1 >: Err, In1 <: In, Out1](that: Handler[R1, Err1, In1, Out1])(implicit trace: Trace): Handler[R1, Err1, In1, Out1]
```

Runs both handlers, keeping only the second output. Also available as the `*>` operator.

### race

```scala
def race[R1 <: R, Err1 >: Err, In1 <: In, Out1 >: Out](that: Handler[R1, Err1, In1, Out1])(implicit trace: Trace): Handler[R1, Err1, In1, Out1]
```

Races two handlers, returning the first to complete.
